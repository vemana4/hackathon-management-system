# Hackathon Management System

[![Python Version](https://img.shields.io/badge/python-3.11%2B-blue)](https://www.python.org) [![Flask](https://img.shields.io/badge/Backend-Flask-green)](https://flask.palletsprojects.com) [![SQLite](https://img.shields.io/badge/DB-SQLite-blue)](https://www.sqlite.org)

The Hackathon Management System is a comprehensive web platform tailored for organizing, managing, and executing large-scale hackathons and competitive bidding events. Built on a robust Flask backend and a lightweight SQLite database, this system provides organizers with a suite of tools to handle participant registration, team formation, and document submissions in collaborative workspaces. What sets this platform apart is its integrated AI-driven auditing services. By utilizing local Isolation Forest machine learning models, the system can automatically detect anomalies in tender bids or project submissions, while integrated Natural Language Processing (NLP) summarizers help judges quickly digest complex project proposals. This ensures a fair, efficient, and technologically advanced environment for all participants.

## 🚀 Key Technologies
- **Web Server**: Flask (Python)
- **Database**: SQLite3
- **AI Services**: Local Isolation Forest models (anomaly detection) and NLP summarizers

## 📦 Getting Started & Installation
```bash
# Install Python dependencies
pip install -r requirements.txt

# Launch the server
python app.py
```
